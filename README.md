# MyBarrelRacers

## Overview
**MyBarrelRacers** is a Razor Pages web application that allows users to manage barrel racing participants and their scores. The app lets you:
- Add new racers.
- View, edit, or delete racer records.
- Search racers by name.

The application also stores additional information about each racer, including:
- Horse Name
- Competition Date
- Event Location
- Qualification status for finals

---

## Features
- **CRUD Operations:** Create, Read, Update, and Delete racer records.
- **Search Functionality:** Search racers by name.
- **Database Integration:** Uses Entity Framework Core with a SQL database.
- **Razor Pages:** Clean, responsive UI using Razor Pages.
- **Validation:** Form validation for required fields and score range.

---

## Requirements
- [.NET 7 SDK](https://dotnet.microsoft.com/download/dotnet/7.0)
- [Visual Studio 2022](https://visualstudio.microsoft.com/) or VS Code with C# extension
- SQL Server or LocalDB for database
